1. Open Adobe Xd
  2. Create a new artboard that is 1200px by 1200px
  3. Create a triangle that is 350w by 500h
  4. Flip upside down (180 degrees) so the point is facing down. It should still be 350x500
  5. Enter the X position as 84 and the Y position as 646
  6. Fill the triangle with the solid hex color #DCA95B and remove the border.
  7. Create a circle that is 350x350
  8. Enter the X position as 84 and the Y position as 432
  9. Fill the circle with the solid hex color #D835A2 and remove the border
  10. Create a new circle that is 250x250
  11. Enter the X position as 350 and the Y position as 125
  12. Fill the circle with the solid hex color #E56D8D and remove the border
  13. Create a new circle that is 150x150
  14. Enter the X position as 718 and the Y position as 175
  15. Fill the circle with the solid hex color #E59DCD and remove the border
  16. Create a new circle that is 50x50
  17. Enter the X position as 963 and the Y position as 400
